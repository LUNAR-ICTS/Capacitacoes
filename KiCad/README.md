| 16h | Curso Introdutório de Projeto de Placas de Circuito Impresso (PCB) — **KiCad**  |                                                                                                                                                                                                              |                                                                                                                                                         |
|     | **A1**                                                                          | Introdução ao projeto de PCBs: o que é PCB, fluxo (esquemático→layout→fabricação→montagem), processos (FR-4, cobre, corrosão, furação, máscara/serigrafia), limitações de fabricação; visão geral do KiCad.  | Criar projeto novo e explorar interface/módulos/arquivos; resolução comentada: organização, nomes/pastas e atalhos iniciais.                            |
|     | **A2**                                                                          | Captura esquemática: o que é esquemático, símbolo vs componente físico, componentes básicos, legibilidade, ERC, relação com fabricação.                                                                      | Capturar esquemático simples (LED + resistor + alimentação); resolução comentada: organização visual, uso de GND/VCC, evitar erros comuns.              |
|     | **A3**                                                                          | Esquemático organizado e preparação p/ layout: organização por blocos, alimentação/terra, footprints, importância do datasheet, símbolo×footprint, checklist pré-PCB.                                        | Associar footprints corretos e preparar o projeto para layout; resolução comentada: critérios de footprint, erros frequentes, validação via datasheet.  |
|     | **A4**                                                                          | Introdução ao layout: camadas (Top/Bottom/Silk/Mask), contorno, placement e erros comuns.                                                                                                                    | Criar contorno e posicionar componentes; resolução comentada: estratégia de placement, alinhamento/espaçamento, uso de grid.                            |
|     | **A5**                                                                          | Roteamento: conceitos básicos, largura/espaçamento (intro), sinal vs alimentação, 45°, plano de terra (GND).                                                                                                 | Roteamento completo; resolução comentada: estratégia alternativa, limpeza visual, quando refazer.                                                       |
|     | **A6**                                                                          | Boas práticas, revisão e montagem: separação sinais/alimentação, montagem (manual/onda/refusão), serigrafia, DRC.                                                                                            | Revisar layout e corrigir erros do DRC; resolução comentada: interpretar DRC, ajustes rápidos, serigrafia “profissional”.                               |
|     | **A7**                                                                          | Arquivos de fabricação: Gerbers, drill, o que a fábrica espera, erros comuns no envio.                                                                                                                       | Gerar Gerbers e conferir visualmente; resolução comentada: checklist pré-envio, evitar retrabalho, conferência rápida.                                  |
|     | **A8**                                                                          | Revisão geral e próximos passos: erros após receber a placa, debug, iteração, evolução (SMD/dupla face/multilayer), caminhos de estudo.                                                                      | Analisar criticamente o projeto final e propor melhorias; resolução comentada: revisão do projeto e melhorias reais.                                    |
